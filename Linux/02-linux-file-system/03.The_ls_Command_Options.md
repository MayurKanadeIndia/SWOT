# Linux `ls` Command — Professional README

---

# Commands — `ls`

The `ls` command is one of the most frequently used Linux commands.

It is used to:

- list files,
- display directories,
- view file details,
- and explore the Linux filesystem.

---

# Syntax

```bash
ls [OPTIONS] [FILES]
```

---

# Basic Concepts

| Symbol | Meaning               |
| ------ | --------------------- |
| `~`    | User's home directory |
| `.`    | Current directory     |
| `..`   | Parent directory      |

---

# 1. Listing the Current Directory

Displays files and directories in the current location.

```bash
ls
```

---

## Listing the Current Directory Explicitly

```bash
ls .
```

---

# 2. Listing Multiple Directories

Displays contents of multiple directories at once.

```bash
ls ~ /var /
```

---

# 3. Long Listing Format — `-l`

Displays detailed information.

Includes:

- permissions,
- owner,
- group,
- size,
- modification date,
- filename.

```bash
ls -l ~
```

---

# 4. Display Hidden Files — `-a`

Displays all files including hidden files.

Hidden files begin with `.`

```bash
ls -la ~
```

---

# 5. Single Column Output — `-1`

Displays one file per line.

```bash
ls -1 /etc
```

---

# 6. Display Directory Information Only — `-d`

Displays information about the directory itself instead of its contents.

```bash
ls -ld /etc
```

---

# 7. Human-Readable Sizes — `-h`

Displays file sizes in:

- KB,
- MB,
- GB,
- TB.

Usually combined with `-l`.

```bash
ls -lh /etc
```

---

# 8. Sort by File Size — `-S`

Displays files sorted by size.

Largest files appear first.

```bash
ls -Sh /var/log
```

---

# Important Note

The `ls` command does not calculate the total size of a directory and all its contents.

Use `du` instead.

```bash
du -sh ~
```

---

# 9. Sort by Extension — `-X`

Displays files sorted alphabetically by extension.

```bash
ls -lX /etc
```

---

# 10. Hide Specific Files — `--hide`

Hides files matching a pattern.

Example:

```bash
ls --hide=*.log /var/log
```

This hides all `.log` files.

---

# 11. Recursive Listing — `-R`

Displays directories recursively.

Shows all nested files and directories.

```bash
ls -lR ~
```

---

# 12. Display Inode Numbers — `-i`

Displays inode numbers for files and directories.

```bash
ls -li /etc
```

---

# Commonly Used `ls` Options

| Option | Purpose                  |
| ------ | ------------------------ |
| `-l`   | Long listing format      |
| `-a`   | Show hidden files        |
| `-h`   | Human-readable sizes     |
| `-S`   | Sort by size             |
| `-X`   | Sort by extension        |
| `-R`   | Recursive listing        |
| `-d`   | Show directory info only |
| `-i`   | Show inode numbers       |
| `-1`   | One file per line        |

# Understanding Hidden Files

Files beginning with `.` are hidden files.

Examples:

```bash
.bashrc
.profile
.gitconfig
```

To display hidden files:

```bash
ls -a
```

---

# Understanding Inodes

An inode is a unique identifier assigned to a file in Linux.

View inode numbers using:

```bash
ls -i
```

---

# Combining Multiple Options

Linux allows combining options together.

Example:

```bash
ls -lah
```

Meaning:

- `-l` → long listing
- `-a` → show hidden files
- `-h` → human-readable sizes
